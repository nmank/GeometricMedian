# Geometric Medians on The Grassmann

Our goal is to create examples for new iteerative algirhtms for calculating a geeeometric median on the Grassmannian. A paper is in the works. This is a library for experimenting with the geometric median on the Grassmannian manifold. It is currently a work in progress

## What's this all about?

The majority of thesee examples are on Gr(1,3).

One example is working towards Gr(2,3) and we are hoping to run a k-medians algorithm on MNIST.
